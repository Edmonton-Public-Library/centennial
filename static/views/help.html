2. Accounts

An account is not required to view content on the TimeMap website. An account is required only if a user wants to upload their own content to the TimeMap website.

Creating a New Account

TO DO - add steps to navigate to create account screen when complete.

A user can create a new account on the Create Account screen. The user must complete the form with the following information:

  • First name
  • Last name
  • Email
  • Username
  • Password
Note that the password must be at least 6 characters long, and the user must confirm the password by entering it twice. The username and password are required so that the user can log back into the website.

The user must also agree to the Terms and Conditions before their account can be created.

Once the user has completed the form, clicking on the "Create Account" button creates a new account for the user:
  • If there are any errors with the user's input (such as leaving a required field empty), error messages are displayed. The user can fix their information and click "Create Account" again to resubmit the form.
  • If the username has already been taken by another user, an error message is displayed. Unfortunately, usernames must be unique. The user must choose another username for their account.
  • If there are no errors with the user's input, a new account is created for the user. A message is displayed to user indicating that an account has been created.
  • The user must verify their email address before they are allowed to login to their new account.
Once a user has created and verified their account, they can upload their own stories to the TimeMap website.

Verifying Your Email Address

A user must verify their email address before they can login to their account for the first time. When a new account is created, an email is sent to the email address that the user provided on the Create Account screen. This email contains a link that the user must follow to validate their account. The user will only be able to login to the TimeMap website once they have verified their email address.

Managing Your Account

TO DO - add steps to navigate to My Account screen when complete.

A user can edit their account details once their have logged into their account, by navigating to the My Account screen. On this screen the user can view and update information about their account, including:

  • First Name
  • Last Name
  • Email
  • Password
For security reasons, the user must enter their current password in order to update their email or their password.

Once the user has updated their information, clicking on the "Save Changes" button will update their account information:
  • If there are any errors with the user's input (such as entering their current password incorrectly), error messages are displayed. The user can fix their information and click "Save Changes" again to resubmit the form.
  • If there are no errors with the user's input, the user's account information is updated. A message is displayed to user indicating that their account has been updated successfully.
